# 운동 데이터 마크다운 가이드

```

├── 06-hip.md       (엉덩이)
└── 07-leg.md       (다리)
```

## ✏️ 마크다운 형식

각 파일은 다음 형식을 따릅니다:

```markdown
# 부위명 (English Name)

부위 ID: 숫자
주요 근육: 근육명들

---

## 난이도: 초급

### 운동명 (English Exercise Name)
- **주요 근육**: 주요 근육
- **보조 근육**: 보조 근육
- **설명**: 운동 설명
- **주의사항**: 주의할 점
- **추천 세트/횟수**: 3세트 × 10-12회
- **기준 무게 (체중 60kg 기준)**: 10-15kg

---

## 난이도: 중급

(위와 동일한 형식)

---

## 난이도: 고급

(위와 동일한 형식)
```

## 🎯 작성 가이드

### 1. 부위 정보
- **부위 ID**: 1~7 (데이터베이스 ID와 매칭)
- **부위명**: 한글명과 영문명 함께 표기
- **주요 근육**: 해당 부위의 주요 근육군

### 2. 난이도 분류
- **초급**: 초보자도 안전하게 할 수 있는 운동
- **중급**: 기본기가 있는 사람을 위한 운동
- **고급**: 숙련자를 위한 고난도 운동

### 3. 운동 정보 필수 항목
- **운동명**: 한글명 + 영문명
- **주요 근육**: 이 운동이 주로 자극하는 근육
- **보조 근육**: 함께 사용되는 보조 근육 (없으면 "-")
- **설명**: 운동 방법을 명확하게
- **주의사항**: 부상 방지를 위한 주의점
- **추천 세트/횟수**: 표준적인 세트/횟수
- **기준 무게**: 체중 60kg 기준 권장 무게

### 4. 기준 무게 작성 예시
- `10-15kg (덤벨 한 개)` - 덤벨 운동
- `30-40kg (바벨 포함)` - 바벨 운동
- `맨몸` - 자중 운동
- `케이블 10-15kg` - 케이블 머신
- `밴드 강도 - 미디엄` - 밴드 운동

## 🔄 수정 방법

### 새로운 운동 추가
1. 해당 부위 마크다운 파일 열기
2. 적절한 난이도 섹션에 운동 추가
3. 위의 형식 그대로 작성
4. 저장하면 자동으로 화면에 반영됨

### 기존 운동 수정
1. 수정할 운동이 있는 마크다운 파일 열기
2. 내용 수정
3. 저장하면 자동으로 화면에 반영됨

## ⚠️ 주의사항

1. **형식 준수**: 마크다운 형식을 정확히 지켜야 파싱이 올바르게 됩니다
2. **필수 항목**: 모든 필수 항목을 빠짐없이 작성하세요
3. **일관성**: 기존 운동의 작성 스타일을 따라주세요
4. **한글/영문**: 운동명은 반드시 한글과 영문을 함께 표기하세요

## 🚀 자동 연동

마크다운 파일을 수정하고 저장하면:
1. Vite의 HMR(Hot Module Replacement)로 자동 리로드
2. `parseExerciseMarkdown()` 함수가 자동 파싱
3. ExerciseSelector 컴포넌트에서 즉시 표시

## 📝 예시

```markdown
### 바벨 벤치 프레스 (Barbell Bench Press)
- **주요 근육**: 대흉근 전체
- **보조 근육**: 삼두근, 삼각근
- **설명**: 벤치에 누워 바벨을 가슴에서 위로 밀어올립니다. 가슴 운동의 왕.
- **주의사항**: 안전바를 설치하거나 보조자와 함께 하세요. 손목 각도 주의.
- **추천 세트/횟수**: 4세트 × 6-8회
- **기준 무게 (체중 60kg 기준)**: 40-50kg (바벨 포함)
```
