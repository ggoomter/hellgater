# 운동 시연 영상 촬영 계획

## 📋 개요

**목적**: 0주차 및 초기 주차에 필요한 기본 운동 시연 영상 제작  
**촬영 기간**: 2주 (2025-12-09 ~ 2025-12-22)  
**목표 영상 수**: 15개 (0-4주차 필수 운동)

---

## 🎯 0주차 필요 운동 목록

### 체력 측정용 운동 (3개)
1. **푸시업 (Push-up)**
   - 길이: 1분 30초
   - 구성: 정면 + 측면 + 슬로우모션
   - 난이도 변형: 무릎 푸시업 포함

2. **플랭크 (Plank)**
   - 길이: 1분
   - 구성: 정면 + 측면
   - 주의사항: 허리 처짐 방지

3. **스쿼트 (Squat)**
   - 길이: 1분 30초
   - 구성: 정면 + 측면 + 후면
   - 난이도 변형: 박스 스쿼트 포함

## 🎬 촬영 사양

### 장비 체크리스트
- [ ] **카메라**: 스마트폰 (1080p 이상) 또는 DSLR
- [ ] **삼각대**: 높이 조절 가능
- [ ] **조명**: 자연광 또는 LED 링라이트 2개
- [ ] **배경**: 단색 벽 (흰색 또는 회색)
- [ ] **의상**: 검은색 또는 진한 회색 운동복
- [ ] **소품**: 요가 매트, 아령, 박스

### 촬영 설정
```yaml
해상도: 1920x1080 (Full HD)
프레임레이트: 60fps (슬로우모션용)
비트레이트: 최소 10Mbps
오디오: 별도 녹음 (후시 더빙)
```

## 📐 촬영 각도 및 구성

### 표준 촬영 구성 (각 운동마다)

#### 1. 정면 각도 (Front View)
```
카메라 위치: 정면 3m 거리
높이: 가슴 높이
용도: 전체 동작 흐름 파악
촬영 시간: 30초 (2-3회 반복)
```

#### 2. 측면 각도 (Side View)
```
카메라 위치: 측면 2.5m 거리
높이: 허리 높이
용도: 자세 정확도 확인
촬영 시간: 30초 (2-3회 반복)
```

#### 3. 슬로우모션 (Slow Motion)
```
카메라 위치: 정면 또는 측면
프레임레이트: 120fps → 30fps 재생
용도: 핵심 구간 강조
촬영 시간: 10초 (1회)
```

#### 4. 흔한 실수 예시 (Common Mistakes)
```
카메라 위치: 정면
용도: 잘못된 폼 시연
촬영 시간: 20초
예시:
  - 푸시업: 허리 처짐, 팔꿈치 각도 틀림
  - 플랭크: 엉덩이 들림, 허리 처짐
  - 스쿼트: 무릎 안쪽으로 모임, 발뒤꿈치 들림
```

## 🎨 후반 작업 (편집)

### 편집 소프트웨어
- **추천**: DaVinci Resolve (무료)
- **대안**: Adobe Premiere Pro, Final Cut Pro

### 편집 구성 (각 영상)

#### 타임라인 구조
```
00:00 - 00:05  인트로
  - 운동 이름 타이틀
  - "헬게이터 운동 가이드" 로고

00:05 - 00:35  정면 시연 (2회)
  - 자막: "정면 뷰"
  - 오버레이: 타겟 근육 하이라이트

00:35 - 01:05  측면 시연 (2회)
  - 자막: "측면 뷰"
  - 오버레이: 호흡 타이밍 아이콘

01:05 - 01:15  슬로우모션
  - 자막: "핵심 구간"
  - 음악: 느려짐

01:15 - 01:35  흔한 실수
  - 자막: "❌ 이렇게 하지 마세요"
  - X 표시 오버레이

01:35 - 01:50  올바른 폼 재확인
  - 자막: "✅ 올바른 자세"
  - 체크 표시 오버레이

01:50 - 02:00  아웃트로
  - 핵심 포인트 3가지 (텍스트)
  - 다음 영상 예고
```

### 오버레이 요소

#### 1. 근육 하이라이트
```
도구: After Effects 또는 Photoshop
방법:
  1. 신체 윤곽 추출
  2. 타겟 근육 부위 색칠 (빨간색 50% 투명도)
  3. 펄스 애니메이션 추가

예시:
  - 푸시업: 가슴, 삼두, 어깨
  - 플랭크: 복근, 코어
  - 스쿼트: 대퇴사두, 둔근
```

#### 2. 호흡 타이밍 아이콘
```
위치: 화면 우측 상단
크기: 80x80px
애니메이션:
  - 들숨: 파란 화살표 ↓ (1초)
  - 날숨: 빨간 화살표 ↑ (1초)
  - 페이드 인/아웃 효과
```

#### 3. 주의사항 텍스트
```
위치: 화면 하단 중앙
폰트: Noto Sans KR Bold, 24pt
배경: 반투명 검은색 박스
색상: 흰색 (중요) / 노란색 (경고)

예시:
  "💡 팁: 코어에 힘을 주세요"
  "⚠️ 주의: 허리가 꺾이지 않도록"
```

## 🎵 오디오 작업

### BGM (배경음악)
```
장르: 업비트 일렉트로닉, 동기부여 음악
템포: 120-130 BPM
볼륨: -20dB (내레이션 방해 안 되게)
출처: Epidemic Sound, Artlist (저작권 프리)
```

### 내레이션 (선택사항)
```
톤: 친근하고 격려하는 목소리
속도: 분당 150-180 단어
스크립트 예시:

"안녕하세요, 헬게이터입니다.
오늘은 푸시업의 올바른 자세를 배워볼게요.
푸시업은 가슴, 삼두, 어깨를 동시에 자극하는
훌륭한 복합 운동입니다.
자, 함께 시작해볼까요?"
```

### 효과음 (SFX)
```
사용 시점:
  - 타이틀 등장: 우웅~ (임팩트)
  - 체크 표시: 띠링! (성공)
  - X 표시: 부웅~ (실패)
  - 전환: 휙~ (스와이프)
```

## 🎬 촬영 팁

### 조명 세팅
```
[조명 1]          [카메라]          [조명 2]
    ↘                 ↓                 ↙
              [촬영 대상]

- 조명 1, 2: 45도 각도, 2m 거리
- 높이: 대상보다 약간 높게
- 밝기: 동일하게 (그림자 최소화)
```

### 카메라 구도
```
푸시업/플랭크:
  - 전신이 프레임 안에
  - 상하 여백: 10%
  - 좌우 중앙 정렬

스쿼트:
  - 발끝부터 머리까지
  - 측면: 무릎 각도 명확히
```

### 촬영 순서
1. 테스트 촬영 (10초)
2. 재생 확인 (조명, 구도, 포커스)
3. 본 촬영 (각 각도 2-3회)
4. 즉시 재생 확인
5. 문제 있으면 재촬영
