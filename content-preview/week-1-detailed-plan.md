# 1주차: 몸을 알다 - 측정의 중요성

## 📋 개요

**주차**: 1주차  
**테마**: "측정하지 않으면 개선할 수 없다"  
**핵심 메시지**: 데이터 기반 운동의 시작  
**예상 소요 시간**: 40-50분  
**완료 보상**: 경험치 +1500, "데이터 수집가" 칭호, 바알시불 레벨 2-3

## 🎬 스토리 시나리오

### Scene 1: 무작정 운동하는 바알시불 (30초)
```
[화면: 바알시불이 열심히 운동 중]

바알시불: "으랏차! 으랏차!"
[무거운 돌을 들려고 하지만 실패]
[다시 시도... 또 실패]

바알시불: "왜 안 되는 거지? 😢"
"어제도 운동했는데... 오늘도 똑같아..."
```

### Scene 2: 현자의 질문 (45초)
```
현자: "바알시불, 질문 하나 하겠네."
"자네는 지금 얼마나 강한가?"

바알시불: "음... 잘 모르겠어요."

현자: "어제보다 오늘 더 강해졌는가?"

바알시불: "그것도... 모르겠어요."

현자: "그렇다면 어떻게 발전을 알 수 있겠는가?"
"측정하지 않으면, 개선할 수 없다네."
```

### Scene 3: 인바디 측정소 탐험 (60초)
```
[화면: 신비로운 측정소 등장]

현자: "이곳은 인바디 측정소."

[인바디 기계 등장, 빛이 번쩍]

바알시불: "우와! 신기해요!"

바알시불: "이 숫자들이 뭐예요?"

현자: "이것이 바로 자네의 현재 상태일세."
"이제부터 이 숫자들이 변하는 것을 보게 될 거야."
```

## 📚 콘텐츠 구성

### 1. 교육 영상: 인바디 지표 완벽 해석 (8분)

#### 파트 1: 체중의 진실 (2분)
```yaml
내용:
  - 체중 ≠ 건강 지표
  - 근육 vs 지방의 무게 차이
  - 같은 체중, 다른 체형 비교 사진

핵심 메시지:
  "체중계 숫자에 집착하지 마세요.
   중요한 건 체성분입니다."

시각 자료:
  - 70kg 근육형 vs 70kg 비만형 비교
  - 근육 1kg vs 지방 1kg 부피 차이
```

#### 파트 2: 체지방률 이해하기 (3분)
```yaml
남성 기준:
  - 3-5%: 보디빌더 대회 수준 (비건강적)
  - 6-13%: 복근 선명, 혈관 보임
  - 14-17%: 운동하는 사람 평균
  - 18-24%: 일반인 평균
  - 25%+: 비만

여성 기준:
  - 10-13%: 보디빌더 대회 수준
  - 14-20%: 운동하는 사람 평균
  - 21-24%: 일반인 평균
  - 25-31%: 약간 과체중
  - 32%+: 비만

시각 자료:
  - 체지방률별 몸 사진 (남/여)
  - 3D 애니메이션: 지방 분포
```

#### 파트 3: 골격근량의 중요성 (2분)
```yaml
내용:
  - 근육 = 대사 엔진
  - 근육 1kg당 하루 13kcal 소모
  - 근감소증 예방

목표 골격근량:
  - 남성: 체중의 40-50%
  - 여성: 체중의 30-40%

계산 예시:
  70kg 남성 → 28-35kg 골격근량 목표
```

#### 파트 4: 기초대사량 (BMR) (1분)
```yaml
정의:
  "아무것도 안 해도 소모되는 칼로리"

영향 요인:
  1. 근육량 (가장 중요)
  2. 나이
  3. 성별
  4. 유전

활용법:
  BMR x 활동계수 = TDEE
  TDEE 기준으로 식단 설정
```

### 3. Before 사진 촬영 완벽 가이드 (영상 5분)

#### 촬영 준비
```yaml
시간:
  - 아침 공복 (기상 후 화장실 다녀온 직후)
  - 매번 같은 시간

복장:
  - 남성: 속옷 또는 반바지
  - 여성: 스포츠 브라 + 레깅스

장소:
  - 단색 벽 앞
  - 밝은 조명 (자연광 최고)
  - 같은 위치 표시 (바닥에 테이프)
```

#### 촬영 각도 (3장 필수)
```
1. 정면
   - 팔 자연스럽게 내림
   - 발 어깨 너비
   - 정면 응시

2. 측면 (왼쪽)
   - 팔 자연스럽게 내림
   - 옆모습 명확히
   - 복부 긴장 풀기

3. 후면
   - 팔 자연스럽게 내림
   - 등/엉덩이 라인 확인
```

#### 촬영 팁
```
✅ DO:
- 타이머 사용 (10초)
- 같은 거리 유지 (2-3m)
- 자연스러운 자세
- 매번 같은 조건

❌ DON'T:
- 거울 셀카 (각도 일정 안 됨)
- 복부에 힘 주기
- 각도 바꾸기
- 조명 바꾸기
```

### 4. 실습 과제

#### 과제 1: 인바디 측정 완료
```yaml
옵션 A: 전문 인바디 측정
  - 헬스장, 보건소, 약국
  - 비용: 무료~5,000원
  - 정확도: ⭐⭐⭐⭐⭐

옵션 B: 가정용 체성분계
  - 샤오미, 유핏 등
  - 비용: 30,000~100,000원
  - 정확도: ⭐⭐⭐⭐

옵션 C: 자가 측정
  - 체중계 + 줄자
  - 비용: 무료
  - 정확도: ⭐⭐⭐

입력 폼:
  체중: [____] kg
  체지방률: [____] % (선택)
  골격근량: [____] kg (선택)
  허리둘레: [____] cm (필수)
  BMR: [____] kcal (선택)

보상: +500 XP
```

#### 과제 2: Before 사진 3장 촬영
```yaml
체크리스트:
  - [ ] 아침 공복 시간 확인
  - [ ] 촬영 장소 준비 (배경, 조명)
  - [ ] 바닥에 위치 표시
  - [ ] 정면 사진 촬영
  - [ ] 측면 사진 촬영
  - [ ] 후면 사진 촬영
  - [ ] 앱에 업로드

보상: +800 XP + "용기있는 시작" 배지
```

#### 과제 3: 목표 체성분 설정
```yaml
계산기 활용:
  현재 → 목표 시뮬레이션

예시:
  현재: 70kg, 체지방률 20%
  목표: 68kg, 체지방률 15% (3개월)

  필요한 변화:
  - 지방 감소: -4.5kg
  - 근육 증가: +2.5kg
  - 순 체중 변화: -2kg

  주간 목표:
  - 지방 -0.35kg/주
  - 근육 +0.2kg/주

보상: +300 XP
```
