### 3. 현재 상태 자가 진단 (10분)

**테스트 항목**:
1. **푸시업 테스트** (1분간 최대 횟수)

   - 6-15개: 중급
   - 16-30개: 상급
   - 31개+: 고급

2. **플랭크 테스트** (최대 유지 시간)
   - 0-30초: 초급
   - 31-60초: 중급

   - 121초+: 고급

3. **스쿼트 테스트** (1분간 최대 횟수)

### 5. 실습 과제 (15분)

#### 과제 1: 인바디 측정 또는 자가 측정
**체크리스트**:
- [ ] 체중 측정 (아침 공복)
- [ ] 허리둘레 측정 (배꼽 위치)
- [ ] 가능하면 인바디 측정

**측정 가이드 영상**: `week0_measurement_guide.mp4`

**입력 폼**:
```
체중: [    ] kg
신장: [    ] cm
허리둘레: [    ] cm

[인바디 데이터 있음]
  체지방률: [    ] %
  골격근량: [    ] kg
  BMR: [    ] kcal
```

#### 과제 2: Before 사진 촬영
**촬영 가이드**:
```
📸 촬영 조건:
- 시간: 아침 공복 (화장실 후)
- 복장: 속옷 또는 운동복
- 조명: 자연광 또는 밝은 실내
- 배경: 단색 벽

📐 촬영 각도:
1. 정면 (팔 자연스럽게 내림)
2. 측면 (왼쪽)
3. 후면

💡 팁:
- 같은 장소, 같은 시간에 촬영
- 거울 셀카보다 타이머 사용 권장
- 얼굴은 가려도 OK
```

**업로드 UI**:
```
[정면 사진 업로드] 📷
[측면 사진 업로드] 📷
[후면 사진 업로드] 📷

⚠️ 사진은 암호화되어 저장되며,
   본인만 볼 수 있습니다.
```

#### 과제 3: 식단 일기 시작
**목표**: 3일간 먹은 모든 음식 기록

**입력 방법**:
1. 사진 촬영 + 업로드
2. 텍스트 입력
3. 음성 메모

**예시**:
```
[2025-12-09 아침]
- 흰쌀밥 1공기
- 김치찌개
- 계란후라이 2개
- 김치

[사진 업로드] 📷

예상 칼로리: 650kcal (자동 분석)
```
