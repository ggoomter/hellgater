# 바알시불 (Baal-Sibul) 캐릭터 디자인 문서

## 🎭 캐릭터 컨셉

### 기본 정보
- **이름**: 바알시불 (Baal-Sibul)
- **종족**: 헬게이트 악마 (Hell Gate Demon)
- **역할**: 사용자의 운동 파트너, 성장 동반자
- **성격**: 순수하고 호기심 많은 어린 악마, 사용자와 함께 배우고 성장

### 네이밍 의미
- **바알 (Baal)**: 고대 신화의 강력한 신, 힘의 상징
- **시불 (Sibul)**: "시작(始)"의 의미, 새로운 출발
- **합성**: "강력함의 시작" - 약한 상태에서 시작해 강해지는 여정

## 🎨 외형 디자인

### 레벨 0: 알 상태 (탄생 전)
```
형태: 달걀 모양
크기: 8cm
색상: 진한 회색, 붉은 균열
특징:
  - 신비로운 빛 발산
  - 맥박처럼 빛이 깜빡임
  - 표면에 고대 문자 각인
```

### 레벨 1-10: 유아기 (0-4주차)
```
형태: 귀여운 작은 악마
크기: 15cm
외형:
  - 큰 둥근 눈 (순수함 표현)
  - 작은 뿔 2개 (아직 무디고 짧음)
  - 작은 날개 (날지 못함, 펄럭거림)
  - 가느다란 꼬리 (감정 표현용)
  - 통통한 몸매 (귀여움 강조)
색상:
  - 피부: 연한 빨강 (#FF6B6B)
  - 눈: 노란색 (#FFD93D)
  - 뿔/꼬리: 진한 빨강 (#C92A2A)
표정:
  - 기본: 😊 호기심 가득
  - 운동 성공: 😄 기쁨
  - 실패: 😢 슬픔
  - 피곤: 😴 졸림
```

### 레벨 11-30: 성장기 (5-12주차)
```
형태: 소년 악마
크기: 30cm
외형:
  - 뿔이 길어지고 날카로워짐
  - 날개가 커짐 (짧은 비행 가능)
  - 근육이 조금씩 생김
  - 꼬리가 길어지고 힘이 생김
색상:
  - 피부: 중간 빨강 (#FA5252)
  - 근육 라인 강조 (어두운 음영)
장비:
  - 작은 아령 (1kg)
  - 헤드밴드
```

### 레벨 31-60: 청년기 (13-20주차) - 5속성 분기
```
기본 형태: 청년 악마
크기: 50cm
공통 특징:
  - 뿔이 더욱 날카롭고 위협적
  - 날개로 자유롭게 비행
  - 근육질 몸매
  - 자신감 넘치는 표정

속성별 외형 변화:

1. 땅(근육) 속성 - 보디빌더형
   색상: 진한 빨강 (#C92A2A)
   특징:
     - 근육이 매우 발달
     - 뿔이 두껍고 강함
     - 바벨 문양 문신
   장비: 파워 벨트, 리프팅 스트랩

2. 불(체력) 속성 - 전사형
   색상: 주황빛 빨강 (#FF6B35)
   특징:
     - 균형잡힌 근육
     - 날개에 불꽃 효과
     - 역동적인 포즈
   장비: 전투 밴드, 글러브

3. 바람(심폐) 속성 - 날렵형
   색상: 하늘빛 (#4ECDC4)
   특징:
     - 날씬하고 유선형
     - 큰 날개
     - 빠른 움직임 잔상
   장비: 러닝화, 심박계

4. 물(지방) 속성 - 균형형
   색상: 푸른빛 (#95E1D3)
   특징:
     - 날씬하고 탄탄함
     - 물방울 효과
     - 우아한 자세
   장비: 요가 매트, 물병

5. 마음(근성) 속성 - 현자형
   색상: 보라빛 (#9775FA)
   특징:
     - 차분한 분위기
     - 눈에서 빛 발산
     - 명상 자세
   장비: 두건, 명상 구슬
```

### 레벨 61-100: 완성기 (21-25주차)
```
형태: 성숙한 악마 전사
크기: 80cm
외형:
  - 완벽한 근육 비율
  - 거대한 날개
  - 왕관 같은 뿔
  - 전신에 빛나는 문양
색상:
  - 속성별 색상 + 금빛 강조
특수 효과:
  - 오라 발산
  - 발걸음마다 빛 효과
  - 주변에 에너지 파동
```

## 😊 표정 시스템

### 기본 표정 (8가지)
1. **기쁨** 😄
   - 눈: 반달 모양
   - 입: 활짝 웃음
   - 사용: 운동 완료, 레벨업

2. **슬픔** 😢
   - 눈: 축 처짐
   - 입: 역 U자
   - 사용: 운동 실패, 목표 미달성

3. **놀람** 😲
   - 눈: 크게 뜸
   - 입: O자
   - 사용: 새로운 발견, 기록 갱신

4. **피곤** 😴
   - 눈: 반쯤 감김
   - 입: 하품
   - 사용: 오버트레이닝 경고

5. **화남** 😠
   - 눈: 찡그림
   - 입: 이 드러냄
   - 사용: 운동 중 힘 낼 때

6. **생각** 🤔
   - 눈: 위를 봄
   - 손: 턱 괴기
   - 사용: 퀴즈, 선택 상황

7. **자랑** 😎
   - 눈: 자신감
   - 포즈: 근육 자랑
   - 사용: 업적 달성

8. **응원** 💪
   - 눈: 반짝임
   - 주먹: 불끈
   - 사용: 사용자 격려

## 🎬 애니메이션

### 기본 동작
1. **Idle** (대기)
   - 제자리에서 가볍게 뛰기
   - 주변 두리번거리기
   - 스트레칭

2. **Walk** (걷기)
   - 통통 튀면서 걷기
   - 꼬리 흔들기

3. **Run** (달리기)
   - 팔 흔들며 전력 질주
   - 날개 펄럭이며 가속

4. **Jump** (점프)
   - 무릎 굽혔다 펴기
   - 날개로 추가 높이

5. **Fly** (비행)
   - 날개 펄럭이기
   - 공중 제자리 비행

### 운동 동작
1. **Push-up** (푸시업)
   - 사용자와 함께 푸시업
   - 힘들어하는 표정

2. **Squat** (스쿼트)
   - 무릎 굽혔다 펴기
   - 중심 잡기

3. **Plank** (플랭크)
   - 떨면서 버티기
   - 땀 흘리기

4. **Cheer** (응원)
   - 팔 흔들며 응원
   - 점프하며 기뻐하기

### 감정 동작
1. **Happy** (기쁨)
   - 빙글빙글 돌기
   - 공중제비

2. **Sad** (슬픔)
   - 고개 숙이기
   - 바닥에 주저앉기

3. **Excited** (흥분)
   - 제자리 뛰기
   - 불꽃 효과
